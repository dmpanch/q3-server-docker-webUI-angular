Windows x86

Two installers, one installs the ioquake3 engine, the other installs Quake 3 game data and provides the latest patch pk3 data from id. Both are required to play Quake 3.
If the installer fails to copy the Quake 3 data (pak0.pk3) from your cd-rom, you can manually do so to your baseq3 directory after installation. The manual method also works for Steam copies of Quake 3.

Mac OS X Universal Binary

Provides latest patch pk3 data from id. You only need to copy the pak0.pk3 file from your legal Quake 3 CD-ROM to complete the installation process.
Installation instructions are in the ReadMe.rtf included in the dmg. An experimental front-end is included for launching ioquake3 and configuring command-line options without invoking Terminal.app.
